# HTML, CSS, JavaScript et Electron.js : Débutant à Expert

**Cursus professionnel e-learning** — Parcours progressif de 320 heures (estimation pédagogique).

---

## Public visé

Développeurs débutants souhaitant devenir autonomes sur le développement Web moderne et la création d'applications Desktop avec Electron.js.

## Compétences visées en fin de parcours

- Développer des applications Web modernes (HTML5, CSS3, JavaScript ES2025+)
- Développer des applications Desktop avec Electron.js
- Persister des données localement avec SQLite
- Connecter une application Electron à une API REST (JWT, offline)
- Tester, déboguer, packager et déployer une application professionnelle

---

## Prérequis

- Aucune expérience en programmation requise
- Ordinateur Windows, macOS ou Linux
- Connexion Internet pour l'installation des outils

## Modalités pédagogiques recommandées

- **70 % pratique / 30 % théorie**
- Projets incrémentaux (chaque module enrichit le précédent)
- Revue de code en fin de chaque module
- Quiz de validation avant passage au module suivant
- Challenge obligatoire pour débloquer le niveau suivant

# Electron.js : Débutant à Expert — wazabiCode

**Cursus professionnel e-learning** — Formation **100 % Electron.js** (~180 heures).

> **Prérequis :** HTML, CSS et JavaScript (non enseignés dans ce cursus).  
> **Plateforme :** ce site e-learning est développé en HTML, CSS et JavaScript.

---

## Objectif

Former un développeur capable de :

- Créer des applications Desktop cross-platform avec Electron.js
- Maîtriser Main Process, Renderer, Preload et IPC
- Intégrer SQLite pour la persistance locale
- Connecter une app Electron à une API REST (JWT, offline)
- Déboguer, tester, packager et déployer une application professionnelle

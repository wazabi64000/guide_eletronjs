Chapitre 5 — Script Preload & contextBridge

Débutant ⏱ 5h Prérequis : Chapitre 4
📚 Preload scripts — Electron

1. Objectifs

  • Créer preload.js
  • Exposer une API sécurisée au Renderer via contextBridge

2. preload.js (ESM)

// preload.js — Pont entre Main et Renderer (ESM)
import { contextBridge } from 'electron';

contextBridge.exposeInMainWorld('electronAPI', {
  platform: process.platform,
  versions: {
    electron: process.versions.electron,
    chrome: process.versions.chrome,
    node: process.versions.node,
  },
});

Explication — preload.js

CodeCe que ça fait (simplement)
import { contextBridge } from 'electron'contextBridge = outil sécurisé pour exposer des données au Renderer.
exposeInMainWorld('electronAPI', …)Crée window.electronAPI accessible dans votre HTML — comme une API globale contrôlée.
platform: process.platformOS actuel : linux, win32 ou darwin (macOS).
process.versions.electronVersion d'Electron utilisée — utile pour le débogage.

3. Configurer main.js

const win = new BrowserWindow({
  webPreferences: {
    preload: path.join(__dirname, 'preload.js'),
    contextIsolation: true,    // Obligatoire (sécurité)
    nodeIntegration: false,    // Obligatoire (sécurité)
  },
});

Explication — webPreferences dans BrowserWindow

CodeCe que ça fait (simplement)
preload: path.join(…)Chemin vers preload.js exécuté avant le chargement de la page.
contextIsolation: trueIsole le code de la page du preload — obligatoire pour la sécurité.
nodeIntegration: falseInterdit au HTML d'accéder directement à Node.js — évite les failles.

4. Utiliser dans index.html

<script>
  document.body.innerHTML += `<p>Electron ${window.electronAPI.versions.electron}</p>`;
</script>

Explication — utilisation dans le Renderer

CodeCe que ça fait (simplement)
window.electronAPIObjet créé par contextBridge dans preload — accessible comme une variable globale.
versions.electronLit la version Electron exposée par le preload — preuve que le pont fonctionne.
Template literal \`…\${}…\`Chaîne JavaScript qui insère une variable dans le HTML.

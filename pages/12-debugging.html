Module 12 — Debugging Electron

Expert ⏱ 8h Prérequis : Module 11

1. Objectifs

  • DevTools pour le Renderer
  • Debug du Main Process (VS Code launch.json)
  • Logs structurés (electron-log)
  • Performance et memory leaks

2. DevTools Renderer

mainWindow.webContents.openDevTools({ mode: 'detach' });

3. Debug Main (VS Code)

// .vscode/launch.json
{
  "version": "0.2.0",
  "configurations": [{
    "name": "Debug Electron Main",
    "type": "node",
    "request": "launch",
    "cwd": "${workspaceFolder}",
    "runtimeExecutable": "${workspaceFolder}/node_modules/.bin/electron",
    "args": ["."],
    "outputCapture": "std"
  }]
}

4. electron-log

import log from 'electron-log';
log.info('App démarrée');
log.error('Erreur IPC', error);

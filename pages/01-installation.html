Chapitre 1 — Installation officielle d'Electron

Débutant ⏱ 3h Prérequis : Chapitre 0
📚 Source officielle : Instructions d'installation avancée — Electron

1. Objectifs

  • Installer Node.js et vérifier npm
  • Installer Electron comme dépendance de développement
  • Configurer correctement package.json (point d'entrée main.js)
  • Comprendre pourquoi npx electron . échoue si main.js n'existe pas encore

2. Installer Node.js

Téléchargez la version LTS sur nodejs.org, puis vérifiez :

node --version   # ex: v22.x.x
npm --version    # ex: 10.x.x

3. Créer le projet (méthode officielle)

La documentation Electron recommande d'installer Electron en dépendance de développement :

mkdir mon-app-electron
cd mon-app-electron
npm init -y
npm install electron --save-dev

Lors du postinstall, npm télécharge le binaire Electron via @electron/get.

4. ⚠️ Corriger package.json (étape obligatoire)

npm init -y crée par défaut "main": "index.js". Or Electron cherche ce fichier pour démarrer — et il n'existe pas encore !

Erreur fréquente :
Cannot find module '.../index.js'. Please verify that the package.json has a valid "main" entry

Solution : modifiez package.json tout de suite après npm init -y : 
// ❌ Après npm init -y (par défaut)
"main": "index.js"

// ✅ Ce qu'il faut mettre (même avant de créer main.js)
"main": "main.js"

Ouvrez package.json et remplacez son contenu par :

{
  "name": "mon-app-electron",
  "version": "1.0.0",
  "description": "Ma première app Electron",
  "main": "main.js",
  "scripts": {
    "start": "electron ."
  },
  "devDependencies": {
    "electron": "^33.0.0"
  }
}

Important : ne lancez pas encore npm start ni npx electron . — le fichier main.js sera créé au chapitre 3, après index.html (chapitre 2).

5. Ordre des étapes (ne pas sauter)

Chapitre 1 npm init + npm install electron + corriger package.json Chapitre 2 créer index.html (interface) Chapitre 3 créer main.js → npm start ✅ première fenêtre

6. Personnalisation (doc officielle)

OptionCommande
Architecture x64 sur ARMnpm install --arch=x64 electron
Plateforme Windows depuis Linuxnpm install --platform=win32 electron
Version alpha/betanpm install electron@beta --save-dev
Sans télécharger le binaire (CI)ELECTRON_SKIP_BINARY_DOWNLOAD=1 npm install

7. Miroir (réseau lent ou Chine)

export ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
npm install electron --save-dev

8. Résolution des problèmes

ErreurCauseSolution
Cannot find module 'index.js'npm init -y a mis main: index.jsChanger en "main": "main.js" et créer main.js (ch. 3)
Unable to find Electron appFichier main absentCréer main.js avant npm start
ELIFECYCLE, ETIMEDOUT, ECONNRESETRéseauRéessayer, miroir, npm install --verbose electron
EACCESSDroits npmCorriger permissions npm
# Voir la progression du téléchargement
npm install --verbose electron

9. Exercice

  1. Créez le dossier mon-app-electron
  2. npm init -y puis npm install electron --save-dev
  3. Modifiez package.json : "main": "main.js" et script "start": "electron ."
  4. Vérifiez node_modules/electron — ne lancez pas encore l'app

10. Challenge

Installez Electron avec --verbose et notez où le binaire est mis en cache sur votre OS (voir doc : ~/.cache/electron/ sur Linux).

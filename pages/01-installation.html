6. npm et package.json

  "name": "mon-projet",
  "version": "1.0.0",
  "type": "module",
  "main": "index.js",
  "scripts": {
    "start": "node index.js",
    "dev": "node --watch index.js"
  },
  "dependencies": {},
  "devDependencies": {}
}

7. Structure d'un projet

mon-projet/ ├── node_modules/ # Dépendances (ne pas versionner) ├── src/ # Code source │ └── index.js ├── .gitignore ├── package.json └── README.md

8. Premier script JavaScript

Créez src/index.js :

// src/index.js — Premier script wazabiCode
const prenom = 'Développeur';
const annee = new Date().getFullYear();

console.log(`Bienvenue ${prenom} sur wazabiCode !`);
console.log(`Année de formation : ${annee}`);

const modules = ['HTML', 'CSS', 'JavaScript', 'Electron'];
modules.forEach((mod, i) => {
  console.log(`Module ${i + 1} : ${mod}`);
});
node src/index.js

9. Exercices

Exercice — Créer un projet

  1. mkdir mon-premier-projet && cd mon-premier-projet
  2. npm init -y
  3. Créer index.js qui affiche votre nom
  4. git init && git add . && git commit -m "Premier commit"

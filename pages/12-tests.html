Chapitre 12 — Tests automatisés

Intermédiaire ⏱ 6h Prérequis : Chapitre 11

1. Objectifs

Tester avant de compiler en .exe / .dmg. Deux niveaux : unitaires et E2E.

2. Tests unitaires — Jest

npm install jest --save-dev

// utils/format.js
export function formaterTitre(t) { return t.trim() || 'Sans titre'; }

// utils/format.test.js
import { formaterTitre } from './format.js';
test('titre vide', () => expect(formaterTitre('')).toBe('Sans titre'));

Explication — test unitaire Jest

CodeCe que ça fait (simplement)
export function formaterTitreFonction pure testée — pas besoin d'Electron pour la tester.
import { formaterTitre } from …Import ESM du fichier à tester.
test('titre vide', …)Définit un cas de test nommé « titre vide ».
expect(…).toBe(…)Vérifie que le résultat est exactement égal à la valeur attendue.
// package.json
"scripts": { "test": "node --experimental-vm-modules node_modules/jest/bin/jest.js" }

Explication — script test Jest ESM

CodeCe que ça fait (simplement)
--experimental-vm-modulesIndispensable pour que Jest comprenne les import ESM.
node_modules/jest/bin/jest.jsLance Jest sans installation globale.

3. Tests E2E — Playwright

npm install @playwright/test --save-dev
npx playwright install

Explication — installer Playwright

CodeCe que ça fait (simplement)
npm install @playwright/test --save-devAjoute Playwright comme outil de test dans le projet.
npx playwright installTélécharge les navigateurs/Chromium nécessaires aux tests E2E.
// tests/electron.spec.js (ESM)
import { test, expect, _electron as electron } from '@playwright/test';

test('fenêtre principale', async () => {
  const app = await electron.launch({ args: ['.'] });
  const window = await app.firstWindow();
  await expect(window).toHaveTitle(/Ma première app/);
  await window.click('#btn');
  await expect(window.locator('#message')).not.toBeEmpty();
  await app.close();
});

Explication — test E2E Playwright

CodeCe que ça fait (simplement)
import { _electron as electron }API Playwright pour lancer et contrôler une app Electron.
electron.launch({ args: ['.'] })Démarre votre app comme en production.
app.firstWindow()Récupère la première fenêtre ouverte.
expect(window).toHaveTitle(…)Vérifie que le titre de la fenêtre correspond.
window.click('#btn')Simule un clic sur le bouton — comme un utilisateur réel.
app.close()Ferme l'application à la fin du test.

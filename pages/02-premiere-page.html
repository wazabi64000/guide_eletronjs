Chapitre 2 — Votre première page HTML

Débutant ⏱ 3h Prérequis : Chapitre 1

1. Objectifs

  • Créer un fichier index.html from scratch
  • Comprendre que l'interface Electron = pages web
  • Préparer le contenu avant de connecter main.js

2. Créer index.html

Dans votre dossier mon-app-electron, créez :

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ma première app Electron</title>
  <style>
    body {
      font-family: system-ui, sans-serif;
      margin: 0;
      padding: 2rem;
      background: #1a1a2e;
      color: #eee;
    }
    h1 { color: #22d3ee; }
    button {
      padding: 0.75rem 1.5rem;
      background: #22d3ee;
      border: none;
      border-radius: 8px;
      cursor: pointer;
      font-size: 1rem;
    }
  </style>
</head>
<body>
  <h1>Bienvenue dans Electron !</h1>
  <p>Ceci est ma première page affichée dans une fenêtre Desktop.</p>
  <button id="btn">Cliquez-moi</button>
  <p id="message"></p>
  <script>
    document.getElementById('btn').addEventListener('click', () => {
      document.getElementById('message').textContent =
        'Bravo ! Votre page JavaScript fonctionne.';
    });
  </script>
</body>
</html>

Explication — index.html ligne par ligne

CodeCe que ça fait (simplement)
<!DOCTYPE html>Indique au navigateur que c'est un document HTML5.
<html lang="fr">Racine de la page ; lang="fr" = français (accessibilité, SEO).
<head>Métadonnées invisibles : titre onglet, encodage, styles.
<meta charset="UTF-8">Encodage des caractères — accents et emojis affichés correctement.
<title>Texte affiché dans la barre de titre de la fenêtre Electron.
<style>…</style>CSS intégré : couleurs, marges, style du bouton.
<body>Contenu visible par l'utilisateur.
<h1>, <p>Titre principal et paragraphe de bienvenue.
<button id="btn">Bouton cliquable ; id permet de le cibler en JavaScript.
<script>…</script>JavaScript exécuté dans le Renderer (comme dans Chrome).
getElementByIdSélectionne un élément HTML par son id.
addEventListener('click')Exécute une fonction quand l'utilisateur clique.
textContent = …Modifie le texte affiché dans l'élément #message.

3. Tester dans le navigateur (optionnel)

Ouvrez index.html dans Chrome pour vérifier le rendu. Ce n'est pas encore Electron, mais utile pour déboguer le HTML/CSS.

4. Structure après le chapitre 2

mon-app-electron/ ├── package.json "main": "main.js" (corrigé ch.1) ├── index.html ✅ créé maintenant ├── node_modules/electron/ └── main.js ❌ chapitre 3 ⛔ npm start → "Cannot find module main.js" → NORMAL
Rappel : index.html ≠ index.js. Le HTML est l'interface ; main.js (pas encore créé) lancera Electron au chapitre 3.

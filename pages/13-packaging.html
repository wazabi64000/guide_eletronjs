Chapitre 13 — Compiler en .exe · .dmg · AppImage

Avancé ⏱ 6h Prérequis : Chapitre 12 (tests OK)

1. Objectifs

  • Installer electron-builder
  • Générer un installateur Windows (.exe), macOS (.dmg), Linux (.AppImage)

2. Installation

npm install electron-builder --save-dev

Explication — installation electron-builder

CodeCe que ça fait (simplement)
npm install electron-builder --save-devTélécharge l'outil qui compile votre app en installateur.
--save-devOutil de build uniquement — pas nécessaire chez l'utilisateur final.

3. package.json

{
  "name": "mon-app-electron",
  "version": "1.0.0",
  "type": "module",
  "main": "main.js",
  "author": "wazabiCode",
  "scripts": {
    "start": "electron .",
    "test": "jest",
    "dist": "electron-builder",
    "dist:win": "electron-builder --win",
    "dist:mac": "electron-builder --mac",
    "dist:linux": "electron-builder --linux"
  },
  "build": {
    "appId": "com.wazabicode.monapp",
    "productName": "Mon App Electron",
    "directories": { "output": "dist" },
    "files": ["**/*", "!tests/**", "!dist/**"],
    "win": {
      "target": "nsis",
      "icon": "assets/icon.ico"
    },
    "mac": {
      "target": "dmg",
      "icon": "assets/icon.icns"
    },
    "linux": {
      "target": "AppImage",
      "icon": "assets/icon.png"
    }
  }
}

Explication — package.json (electron-builder)

CodeCe que ça fait (simplement)
"dist": "electron-builder"Script npm qui compile l'app en installateur (.exe, .dmg, .AppImage).
"build".appIdIdentifiant unique de l'app (ex : style nom de domaine inversé).
productNameNom affiché à l'utilisateur dans le menu et l'installateur.
directories.output: "dist"Dossier où les installateurs compilés seront créés.
win.target: "nsis"Windows : installateur classique .exe avec assistant.
mac.target: "dmg"macOS : image disque .dmg.
linux.target: "AppImage"Linux : fichier unique exécutable sans installation.

4. Workflow recommandé

1. npm test ← tests passent 2. npm start ← test manuel 3. npm run dist ← génère dist/ 4. Installer et tester le .exe / .dmg / .AppImage

5. Commandes

npm run dist         # OS actuel
npm run dist:win     # Windows (depuis n'importe quel OS avec Wine si Linux)
npm run dist:mac     # macOS (nécessite un Mac pour signer)
npm run dist:linux   # Linux AppImage

Explication — commandes de compilation

CodeCe que ça fait (simplement)
npm run distCompile pour l'OS sur lequel vous travaillez (Windows, macOS ou Linux).
npm run dist:winForce la génération d'un installateur Windows (.exe).
npm run dist:macForce un .dmg — signature Apple requiert souvent un Mac.
npm run dist:linuxForce un fichier .AppImage exécutable sur Linux.

6. Résultat dans dist/

7. Exercice

Ajoutez une icône dans assets/, lancez npm run dist, installez votre propre application.

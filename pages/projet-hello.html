Projet 1 — Hello Electron (packagé)

Débutant ⏱ 4h Prérequis : Chapitres 1 à 3

Objectif

Application minimale : une page, un bouton, compilée en installateur.

Étapes (dans l'ordre)

  1. npm init -y → vous obtenez "main": "index.js"
  2. npm install electron --save-dev
  3. Corriger package.json : "main": "main.js" + "start": "electron ."
  4. Supprimer index.js s'il existe (rm index.js)
  5. Créer index.html (chapitre 2)
  6. Créer main.js (chapitre 3)
  7. npm start — vérifier la fenêtre
  8. npm run dist — installateur (chapitre 13)

package.json final attendu (ESM)

{
  "name": "mon-app-electron",
  "version": "1.0.0",
  "type": "module",
  "main": "main.js",
  "scripts": {
    "start": "electron ."
  },
  "devDependencies": {
    "electron": "^42.5.0"
  }
}

Explication — package.json (champs importants)

CodeCe que ça fait (simplement)
"name"Nom de votre projet. Utilisé par npm pour l'identifier.
"version"Numéro de version de l'app (ex : 1.0.0 → 1.0.1 pour une mise à jour).
"type": "module"Active les modules ESM : dans vos .js, vous écrivez import au lieu de require().
"main": "main.js"Fichier que Electron exécute en premier au lancement — le Main Process.
"scripts".startDéfinit la commande npm start → lance electron . (le point = dossier courant).
devDependenciesDépendances pour le développement seulement (pas livrées telles quelles à l'utilisateur).
"electron"Le package Electron : contient Chromium + Node.js embarqués pour votre app Desktop.

Tous les .js du Main et Preload utilisent import / export.

Livrable

Dossier dist/ avec l'installateur + capture d'écran.

Projet 1 — Hello Electron (packagé)

Débutant ⏱ 4h Prérequis : Chapitres 1 à 3

Objectif

Application minimale : une page, un bouton, compilée en installateur.

Étapes

  1. npm init -y → corriger "main": "main.js" dans package.json
  2. npm install electron --save-dev
  3. Créer index.html puis main.js (dans cet ordre)
  4. npm start — vérifier la fenêtre
  5. npm run dist — obtenir l'installateur

Livrable

Dossier dist/ avec l'installateur + capture d'écran.

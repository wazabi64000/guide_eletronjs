Chapitre 9 — Ouvrir & enregistrer des fichiers

Intermédiaire ⏱ 5h Prérequis : Chapitre 8

1. Dialog Ouvrir / Enregistrer (ESM — dans main.js)

import { dialog, ipcMain } from 'electron';
import fs from 'node:fs';

ipcMain.handle('dialog:open', async () => {
  const { canceled, filePaths } = await dialog.showOpenDialog({
    properties: ['openFile'],
    filters: [{ name: 'Texte', extensions: ['txt', 'md'] }],
  });
  if (canceled) return null;
  return fs.readFileSync(filePaths[0], 'utf-8');
});

ipcMain.handle('dialog:save', async (_e, contenu) => {
  const { canceled, filePath } = await dialog.showSaveDialog({
    filters: [{ name: 'Texte', extensions: ['txt'] }],
  });
  if (canceled) return false;
  fs.writeFileSync(filePath, contenu, 'utf-8');
  return true;
});

Explication — ouvrir / enregistrer fichiers

CodeCe que ça fait (simplement)
dialog.showOpenDialogOuvre l'explorateur « Ouvrir un fichier » natif du système.
properties: ['openFile']Mode sélection d'un fichier (pas un dossier).
filters: [{ extensions: ['txt'] }]Limite les types visibles : fichiers .txt et .md.
canceledtrue si l'utilisateur a annulé — on ne fait rien.
filePaths[0]Chemin complet du fichier choisi par l'utilisateur.
dialog.showSaveDialogOuvre la fenêtre « Enregistrer sous ».
filePathChemin où l'utilisateur veut sauvegarder.

2. preload.js (ESM)

import { contextBridge, ipcRenderer } from 'electron';

contextBridge.exposeInMainWorld('api', {
  ouvrir: () => ipcRenderer.invoke('dialog:open'),
  enregistrer: (c) => ipcRenderer.invoke('dialog:save', c),
});

Explication — preload pour fichiers

CodeCe que ça fait (simplement)
ouvrir: () => ipcRenderer.invoke(…)Expose window.api.ouvrir() — ouvre le dialogue système via le Main.
enregistrer: (c) => …Passe le contenu c au Main pour la boîte « Enregistrer sous ».
